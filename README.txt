Blog Manager Russian language files
===================================

Russian translation for "Blog Manager" OpenCart module.

Note: I'm not the author of the module itself, it is commercial product and can be bought from "OpenCart Extensions" catalog.

All product and company names are trademarks™ or registered® trademarks of their respective holders. Use of them does not imply any affiliation with or endorsement by them.


Requirements
------------

 - OpenCart 1.5.6
   http://www.opencart.com/index.php?route=download/download

 - OpenCart Blog Manager module
   http://www.opencart.com/index.php?route=extension/extension/info&extension_id=4552

Installation
